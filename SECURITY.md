# Security

## 脅威モデル（軽量）
守る対象:
- プロンプトと入力データ。
- `worklogs/` に残るログ。
- 対象リポジトリのソースコード。

主なリスク:
- ログへの秘密情報の混入。
- Codex 実行中の誤操作。
- worktree の不要な残存。

## 秘密情報の取り扱い
- プロンプトと入力に秘密を入れないでください。
- `.env` の自動読み込みは実装していません。
- 必要なら実行前に手動でマスキングします。

## ログのマスキング規約
- 自動マスキングは行いません。
- 共有前に人手で秘匿化します。
- 共有対象は最小限にします。

## 脆弱性報告
- 現時点の専用窓口は不明です。
- GitHub Issue で報告してください。
- 秘密情報は記載しないでください。
- 緊急連絡先は別途確保が必要です。

## 保証
- 秘密情報の自動保護は行いません。

## 仮定
- 利用者が秘密情報を扱う運用を整備します。
