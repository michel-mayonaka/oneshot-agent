「Animal Tower Battle」に近い、画像アセットのみ（アニメーションなし）のブラウザ専用ゲームを、このカレントディレクトリ直下に作成してください。

技術的制約:
- HTML + Canvas + JavaScript を使用する。
- 物理演算には、CDNから読み込む Matter.js を使用する（ビルドツール不要）。
- 生成するファイルは次のとおり: index.html, style.css, game.js, animals.json, README.md。
- アセットのパスは ./assets/animals/*.png（実在する画像は生成しない。画像が無い場合は色付きのプレースホルダーで代用する）。

ゲームプレイ:
- 画面上部中央に動物を1体スポーンする。プレイヤーはマウス移動で左右に動かせる。
- クリックまたは Space キーで落下させる。落下後、動物が安定（sleep状態）したら次の動物をスポーンする。
- まずは動物ごとに animals.json で定義した、単純な当たり判定形状（円または矩形）を使う。
- 床と左右の壁を追加する。
- いずれかの動物が「失敗ライン」より下へ落ちたらゲームオーバーとし、リスタートできるようにする。
- スコアは「積み上げに成功した動物の数」。次に出る動物のプレビューを表示する。

堅牢性:
- 画像が存在しない場合でも落ちないようにする（動物名を表示した、角丸の色付き矩形を描画する）。
- 反発で暴れないように、角速度の最大値をクランプし、摩擦/反発/密度などの値を妥当に設定する。
- 「Reset」ボタンと、Rキーによるリセットを追加する。

README:
- ローカルでの起動方法として「python -m http.server」でサーバを起動してブラウザで開く手順を書く。
